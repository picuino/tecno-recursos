LibreCAD¶

LibreCAD is a free code program to make designs in 2 dimensions. It is a development derived from the QCAD v2 code and is maintained by the free software community.

QCAD¶

QCAD is a free code program up to version 2. As of version 3, the proprietary company distributes it as a paid program. It is the precursor program of LibreCAD.
