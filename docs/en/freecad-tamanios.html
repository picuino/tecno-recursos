4. Pieces size¶

In this tutorial we are going to learn how to change the size of the simple pieces to form a different piece from the standard pieces.


  1. Open the FreeCAD application and click on the icon to create a new document .

    A new tab will open with an empty document, where we can start designing.


  2. We select the Part workbench to start designing objects in 3 dimensions.


  3. At this point we are going to add the reference axes to help us place the pieces correctly.

    In the menu View... Activate or deactivate the cross of the axes.

    In English View... Toggle axis cross


  4. Now we create a cube by clicking the first icon on the solid object bar.

    We select to see the piece in isometric view.

    The piece will look like the following image.


  5. We select the cube and in the Data tab we open the Box menu where we will see the Length, Width and Height of the cube (Length, Width, Height).

    In this menu we can modify the size of the cube in the x, y, z axes.


  6. Cube sizes can be changed by typing in a new number or by clicking on the number and rolling the mouse wheel.

    By moving the mouse wheel you can see how the cube changes in size on the screen.
