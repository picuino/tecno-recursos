******************
 Tutorial de HTML
******************

Tutorial de HTML para crear páginas web.

El código HTML se utiliza para almacenar el contenido de las
páginas web. Es un lenguaje basado en **<etiquetas>** que
identifican las diferentes secciones de un documento.

En las siguientes prácticas se verán una a una las etiquetas
más importantes del lenguaje HTML y cómo se utilizan para
crear páginas web.
