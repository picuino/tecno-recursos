2. Potenciómetro¶

Un potenciómetro es un dispositivo que permite medir desplazamientos o giros. Un potenciómetro giratorio, como el de la imagen siguiente, puede medir ángulos.

Un potenciómetro lineal, como el de la imagen siguiente, puede medir desplazamientos lineales.

Estos componentes se utilizan también para introducir rápidamente valores en un sistema digital. Ejemplos de esto serían los controles de sonido, variadores de intensidad de luz, etc.
