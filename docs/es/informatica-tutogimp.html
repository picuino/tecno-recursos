Tutorial de Gimp¶

Tutorial para aprender a manipular imágenes de mapa de bits con el programa de código libre Gimp, en Castellano.

Basado en tecnología Flash emulada con Ruffle.

En este enlace puedes descargar todas las imágenes de las prácticas en un solo archivo zip.
