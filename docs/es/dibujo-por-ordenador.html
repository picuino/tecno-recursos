Diseño asistido por ordenador¶

LibreCAD¶

LibreCAD es un programa de código libre para realizar diseños en 2 dimensiones. Es un desarrollo derivado del código de QCAD v2 y está mantenido por la comunidad de software libre.

Prácticas y manual de LibreCAD.

QCAD¶

QCAD es un programa de código libre hasta la versión 2. A partir de la versión 3 la empresa propietaria lo distribuye como programa de pago. Es el programa precursor de LibreCAD.
