Dibujar disfraces
=================

.. image:: scratch/_images/scratch-p06-06.png
   :width: 240px


En esta práctica vamos a programar un juego que consiste en
dibujar diferentes personajes por la pantalla con diferentes
tamaños, giros y posiciones.

|br|

1. Iniciamos el |editor_de_Scratch|.

   |br|

#. En la pestaña disfraces, borramos uno de los gatos y añadimos
   varios personajes más.

   .. image:: scratch/_images/scratch-p06-disfraces.png

   |br|

#. A continuación programamos las siguientes instrucciones.

   .. image:: scratch/_images/scratch-p06-prog1.png
      :width: 550px

   .. image:: scratch/_images/scratch-p06-prog2.png
      :width: 317px

   .. image:: scratch/_images/scratch-p06-prog3.png
      :width: 338px

   .. image:: scratch/_images/scratch-p06-prog4.png
      :width: 284px

   |br|

#. En este momento podemos jugar con la pantalla a dibujar
   personajes con los siguientes controles.

   * Flecha arriba = Más tamaño

   Flecha abajo = Menos tamaño

   Flecha derecha = Girar a la derecha el disfraz

   Flecha izquierda = Girar a la izquierda el disfraz

   Tecla Espacio = Cambiar el disfraz a otro personaje

   Clic con el ratón = Hacer copia del disfraz en la pantalla

   |br|


-----


Ejercicios
----------

1. Realiza un dibujo artístico en el que aparezcan todos los
   disfraces en algún escenario.

#. Añade nuevos disfraces para poder dibujar también otros
   objetos.

#. Modifica el programa para que al sellar el disfraz
   (clic del ratón) aparezcan seis imágenes del disfraz
   girados cada uno 60 grados.

#. Modifica el programa para que al sellar el disfraz
   (clic del ratón) aparezcan cuatro imágenes del disfraz
   del tamaño original, un 50%, un 25%
