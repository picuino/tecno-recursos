Copiar formato
==============

En este ejercicio vamos a usar la herramienta
"Copiar formato" para realizar modificaciones de
formato con mayor facilidad.

|br|

1. Primero descargamos y abrimos con LibreOffice Writer
   el documento de ejemplo.

   :download:`Ejercicio de copiar formato
   <writer/writer-copiar-formato.odt>`

   |br|

#. A continuación vamos a seleccionar la primera palabra
   con el ratón y cambiaremos su formato a color verde
   negrita de 14pt de tamaño.

   Recuerda que para cambiar el formato de una palabra debemos
   elegir en el menú ``Formato... Carácter``.

   .. image:: writer/_images/writer-p21-imagen01.png
           :align: center

   |br|

#. Una vez cambiado el formato y con la palabra todavía
   seleccionada, vamos a pulsar en el botón de copiar formato
   |button-copiar-formato|.

   A continuación seleccionamos otra palabra con el pincel de
   formato para que se copie el formato.

   .. image:: writer/_images/writer-p21-imagen02.png
           :align: center

   |br|

#. Si queremos copiar el formato varias veces, debemos pulsar
   dos veces en el pincel de formato y en ese caso el pincel
   seguirá activo hasta que pulsemos la tecla escape
   |tecla-escape|.

   Pulsa dos veces en el botón Copiar formato y copia el formato
   en ocho palabras del texto. Cuando termines presiona la
   tecla escape |tecla-escape|.

   .. image:: writer/_images/writer-p21-imagen03.png
           :align: center

   |br|
