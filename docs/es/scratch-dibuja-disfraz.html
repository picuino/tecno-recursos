3. Dibujar disfraces¶

En esta práctica vamos a programar un juego que consiste en dibujar diferentes personajes por la pantalla con diferentes tamaños, giros y posiciones.


  1. Iniciamos el editor de Scratch.


  2. En la pestaña disfraces, borramos uno de los gatos y añadimos varios personajes más.


  3. A continuación programamos las siguientes instrucciones.


  4. En este momento podemos jugar con la pantalla a dibujar personajes con los siguientes controles.

    • Flecha arriba = Más tamaño

    Flecha abajo = Menos tamaño

    Flecha derecha = Girar a la derecha el disfraz

    Flecha izquierda = Girar a la izquierda el disfraz

    Tecla Espacio = Cambiar el disfraz a otro personaje

    Clic con el ratón = Hacer copia del disfraz en la pantalla


Ejercicios¶

  1. Realiza un dibujo artístico en el que aparezcan todos los disfraces en algún escenario.
  2. Añade nuevos disfraces para poder dibujar también otros objetos.
  3. Modifica el programa para que al sellar el disfraz (clic del ratón) aparezcan seis imágenes del disfraz girados cada uno 60 grados.
  4. Modifica el programa para que al sellar el disfraz (clic del ratón) aparezcan cuatro imágenes del disfraz del tamaño original, un 50%, un 25%
