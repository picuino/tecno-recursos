3. Sensor crepuscular¶

En esta práctica vamos a realizar un sensor de luz con una resistencia LDR y un transistor NPN.

La resistencia es el sensor que recibe la luz y la convierte en corriente. Cuanto mayor sea el nivel de luz, mayor corriente pasará por la resistencia LDR.

El transistor recibe esa corriente y la amplifica para iluminar un led con una corriente mayor.

La ficha para realizar la actividad tiene por una cara el esquema sin terminar, para que se complete a lápiz. Por la otra cara está el esquema completo ya terminado.
