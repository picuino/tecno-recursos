Etiqueta span
=============
``<span class="">  </span>``
   La etiqueta span permite asignar un estilo a un **elemento
   pequeño** dentro de una etiqueta de bloque.

   Por ejemplo, asigna un color a una palabra dentro de un
   párrafo.

   ``<p> Frase con <span class="Azul"> color azul </span> </p>``
